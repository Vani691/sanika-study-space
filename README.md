# Sanika’s Study Space

Sanika’s Study Space is a polished, responsive study planner web app created as a thoughtful, personalized project for a friend named Sanika. It combines a calm pastel interface, a focus timer, task organization tools, and local persistence so the experience feels personal, elegant, and practical.

## Project Overview

This application helps Sanika plan study sessions, track progress, and stay focused with a gentle, premium-feeling dashboard. Everything runs entirely in the browser using React, Vite, JavaScript, and localStorage, with no backend required.

## Features

- Responsive dashboard with total, completed, pending, and overdue task summaries
- Circular progress indicator with dynamic motivational messaging
- Beautiful glassmorphism task creation form
- Task editing, completion toggling, and delete confirmation flow
- Search, priority filtering, and sorting options
- Focus section with a Pomodoro timer for 25-minute focus and 5-minute break cycles
- Random inspirational quote card with quick refresh action
- Light/dark theme toggle saved in localStorage
- Sample tasks loaded on first launch, with the option to clear them all
- Accessible labels, semantic layout, and empty states for a smoother experience

## Technologies Used

- React
- Vite
- JavaScript
- Lucide React icons
- Modern CSS with glassmorphism styling
- localStorage for persistence

## Installation

1. Clone the repository.
2. Open the project folder.
3. Install dependencies:

```bash
npm install
```

## Commands to Run

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
